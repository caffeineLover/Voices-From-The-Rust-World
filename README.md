# Voices from the Rust World

Vintage Story mod for human-recorded narration of lore books and scrolls.

The project is split into:

- `voicesfromtherustworld`: framework/code mod
- `vfrwdefaultvoices`: bundled default content-only voice pack

See [documentation/README.md](documentation/README.md) for setup,
debugging, voice-pack structure, and recording workflow.
